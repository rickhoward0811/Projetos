
**Repositório acadêmico criado para reunir e apresentar todos os projetos
desenvolvidos ao longo do curso**, com foco no aprendizado de lógica de programação
e desenvolvimento de aplicações.

Finalidade Acadêmica

Este repositório foi desenvolvido para **apresentação na faculdade**, funcionando
como um portfólio acadêmico dos projetos realizados durante as disciplinas do curso.

Aprendizados Gerais

Ao longo do desenvolvimento dos projetos presentes neste repositório, foram
aplicados e aprofundados diversos conceitos, principalmente:

* Lógica de programação
* Estruturas condicionais (if / else)
* Tomada de decisões e validações
* Organização do raciocínio lógico
* Resolução de problemas

Durante esse processo, enfrentei **dificuldades reais na construção da lógica**,
o que contribuiu significativamente para a evolução do meu aprendizado.
Todos os projetos foram **desenvolvidos de forma individual**.
