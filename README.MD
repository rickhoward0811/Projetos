# Abruto  
**Repositório Acadêmico**

Repositório criado para reunir e apresentar todos os projetos desenvolvidos ao longo do curso, com foco no aprendizado de lógica de programação e desenvolvimento de aplicações.

---

##  Sobre  
Este repositório foi desenvolvido para apresentação na faculdade, funcionando como um portfólio acadêmico dos projetos realizados durante as disciplinas do curso.

---

##  Aprendizados  
Ao longo do desenvolvimento dos projetos, foram aplicados e aprofundados diversos conceitos:

- Lógica de programação  
- Estruturas condicionais (if / else)  
- Tomada de decisões e validações  
- Organização do raciocínio lógico  
- Resolução de problemas  

Todos os projetos foram desenvolvidos de forma individual, enfrentando dificuldades reais que contribuíram significativamente para minha evolução.
